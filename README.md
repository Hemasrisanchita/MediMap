# 🚑 LifeLine Locator

A smart and user-friendly web application to find nearby hospitals, available beds, and ambulances in real-time.

## 🌟 Problem
During emergencies (like COVID-19), people struggle to quickly find available hospital beds and ambulance services.

## 💡 Solution
LifeLine Locator provides a centralized platform to:
- View nearby hospitals on a map
- Check availability of ICU, Oxygen, and General beds
- Access emergency contact details instantly

## 🚀 Features
- 🗺️ Map-based hospital view
- 🔍 Smart filters (ICU / Oxygen / General beds)
- 🚑 Ambulance availability display
- 📞 One-click emergency contact
- ⚡ Fast and responsive UI

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Maps: OpenStreetMap / Leaflet (or mock map)
- Deployment: GitHub Pages / Vercel / Netlify

## ⚙️ How it works
- Uses mock data to simulate real-time availability
- Displays hospitals dynamically on map
- Filters refine results based on user needs

## 🎯 Impact
- Saves critical time during emergencies
- Improves accessibility to healthcare resources
- Inspired by real-world crisis situations

## 🔮 Future Scope
- Real-time API integration with hospitals
- AI-based hospital recommendation
- Live ambulance tracking
- Admin dashboard for hospital updates
